================================================================================
  Сборка APK (Android) для Mortgage Calculator (Kivy + KivyMD + Buildozer)
================================================================================

Официальная документация Kivy: Packaging / Buildozer for Android
  https://kivy.org/doc/stable/guide/packaging-android.html

Buildozer рассчитан на Linux. На Windows используйте WSL2 (Ubuntu) или
виртуальную машину с Ubuntu (как в уроке).

--------------------------------------------------------------------------------
1. Система: Ubuntu 22.04+ (или WSL2 Ubuntu)
--------------------------------------------------------------------------------

Установите зависимости для сборки (пакеты из документации Buildozer; набор
может слегка отличаться по версии дистрибутива):

  sudo apt-get update
  sudo apt-get install -y --no-install-recommends \
    git zip unzip openjdk-17-jdk python3-pip python3-venv \
    autoconf automake libtool pkg-config zlib1g-dev libncurses5-dev \
    libncursesw5-dev libtinfo5 cmake libffi-dev libssl-dev \
    build-essential patch libc6-dev

(При необходимости см. актуальный список в документации Buildozer / Kivy.)

--------------------------------------------------------------------------------
2. Клонирование репозитория и виртуальное окружение
--------------------------------------------------------------------------------

  git clone <URL-репозитория>
  cd MortgageCalculator
  python3 -m venv .venv
  source .venv/bin/activate          # Windows WSL: то же самое
  pip install --upgrade pip
  pip install "cython<3.1" buildozer
  pip install -r requirements.txt

--------------------------------------------------------------------------------
3. Ресурсы для иконки и splash (обязательно перед первой сборкой)
--------------------------------------------------------------------------------

В buildozer*.spec указаны файлы в data/logo/. Сгенерируйте минимальные PNG:

  python3 scripts/generate_logo_assets.py

--------------------------------------------------------------------------------
4. Файл спецификации Buildozer
--------------------------------------------------------------------------------

В репозитории уже есть:

  buildozer.spec           — по умолчанию 32-bit (armeabi-v7a)
  buildozer-32-bit.spec    — 32-bit APK
  buildozer-64-bit.spec    — 64-bit APK (arm64-v8a)

Для локальной сборки скопируйте нужный файл:

  cp buildozer-32-bit.spec buildozer.spec
  # или для 64-bit:
  # cp buildozer-64-bit.spec buildozer.spec

Проверьте в [app]: title, package.name, package.domain, version, requirements,
orientation (в проекте: portrait + fullSensor для манифеста).

Точка входа приложения: main.py (в спеках задано source.main).

--------------------------------------------------------------------------------
5. Первая инициализация (если buildozer.spec создаёте с нуля)
--------------------------------------------------------------------------------

Если спека ещё нет:

  buildozer init

В этом проекте спеки уже настроены — init не обязателен.

--------------------------------------------------------------------------------
6. Сборка debug APK
--------------------------------------------------------------------------------

  python -m buildozer android debug

Первая сборка скачивает Android SDK/NDK и занимает много времени и места на
диске. Готовый APK:

  bin/*.apk

--------------------------------------------------------------------------------
7. Установка на телефон
--------------------------------------------------------------------------------

Скопируйте APK на устройство (USB, облако, почта), откройте файл на Android,
разрешите установку из неизвестных источников при необходимости, установите.

--------------------------------------------------------------------------------
8. KivyMD 2.x
--------------------------------------------------------------------------------

В requirements.txt указано kivymd>=2.0.0. Если pip ставит устаревшую ветку и
сборка падает, в buildozer.spec в строке requirements можно указать URL
архива KivyMD с GitHub (см. документацию pip/buildozer).

--------------------------------------------------------------------------------
9. CI
--------------------------------------------------------------------------------

Сборка в облаке настроена в .github/workflows/android-build.yml и .travis.yml
(копирование нужного spec, затем python -m buildozer android debug).
